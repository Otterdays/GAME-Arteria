# UI Audit — 2026-03-08

## Issue: Skills Panel TextInput ReferenceError
**Status:** Fixed (v0.6.1)

### Description
The search bar introduced in v0.6.0 was using the `TextInput` component from `react-native`, but the component was not imported at the top of the file. This caused the app to crash when navigating to the Skills tab or when the component was rendered.

### Technical Detail
- **File:** `apps/mobile/app/(tabs)/skills.tsx`
- **Error:** `ReferenceError: Can't find variable: TextInput`
- **Fix:** Added `TextInput` to the `react-native` import list.

### Audit of Other Tabs
- **Bank Tab:** Verified `TextInput` is correctly imported.
- **Settings Tab:** Verified `TextInput` is correctly imported.
- **Combat Tab:** Verified no missing imports detected in current alpha version.
- **Stats Tab:** Verified no missing imports.
