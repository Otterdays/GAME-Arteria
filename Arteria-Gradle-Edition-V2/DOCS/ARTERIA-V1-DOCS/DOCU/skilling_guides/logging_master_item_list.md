# Logging (Woodcutting) Master Item List

> **Purpose:** An exhaustive dictionary of all trees, logs, tools, and secondary drops associated with the Logging skill tree, compiled from OSRS, Melvor Idle, Curse of Aros, and CherryTree, and adapted for Arteria's themes.

---

## 1. Trees & Logs (Primary Resources)
*The core progression of gatherable wood. Used directly in Firemaking, Fletching, and Construction.*

- **Pine Tree** -> Pine Log
- **Evergreen Tree** -> Evergreen Log
- **Dead Tree** -> Dead Log *(Useful for early game Firemaking only)*
- **Birch Tree** -> Birch Log
- **Oak Tree** -> Oak Log
- **Apple Tree** -> Applewood *(Secondary drop: Apples)*
- **Willow Tree** -> Willow Log
- **Teak Tree** -> Teak Log
- **Maple Tree** -> Maple Log
- **Ash Tree** -> Ash Log
- **Mahogany Tree** -> Mahogany Log
- **Chestnut Tree** -> Chestnut Log *(Secondary drop: Chestnuts)*
- **Yew Tree** -> Yew Log
- **Eucalyptus Tree** -> Eucalyptus Log
- **Magic Tree** -> Magic Log
- **Redwood Tree** -> Redwood Log
- **Corundum Tree** -> Corundum Log *(Crystal tier)*
- **Aetherwood Tree** -> Aether Log *(Cosmic tier)*

---

## 2. Tools (Axes / Hatchets)
*Equipped to chop trees. Higher tier axes chop faster and can tackle harder woods.*

- **Bronze Axe**
- **Iron Axe**
- **Steel Axe**
- **Mithril Axe**
- **Adamant Axe**
- **Rune Axe**
- **Dragon Axe / Crimson Hatchet**
- **Crystal Axe** *(Requires Crystal Shards to maintain)*
- **Infernal Axe / Flame-forged Axe** *(Burns logs automatically for Firemaking XP)*
- **Hatchet of the Gods / Aether Axe** *(Cosmic endgame tool. Doubles log drops).*

---

## 3. Secondary Drops & Byproducts
*Items that randomly drop while logging. These bridge Logging to Farming, Crafting, and Cooking.*

- **Bird's Nest (Empty)** -> Crushed for Herblore/Alchemy.
- **Bird's Nest (Seed)** -> Contains a random Tree Seed (Pine Seed, Magic Seed, etc.).
- **Bird's Nest (Ring)** -> Contains a random low-tier ring (Gold ring, Sapphire ring).
- **Bird's Nest (Egg)** -> Contains a random egg (for Summoning or Pets).
- **Geode** -> Drops from Redwood/Crystal trees. Contains gems.
- **Amber Fragment** -> Drops from ancient trees (Yew, Magic). Used in crafting amulets.
- **Apples / Chestnuts / Cherries** -> Dropped by specific fruit-bearing trees. Used in Cooking.

---

## 4. Refined Wood (Planks & Materials)
*Logs are taken to a Sawmill or Engineering bench to be refined into usable building materials.*

- **Pine Plank** (1 Pine Log + 1 Nail + Coins)
- **Oak Plank** (1 Oak Log + 2 Nails + Coins)
- **Teak Plank** (1 Teak Log + 3 Nails + Coins)
- **Mahogany Plank** (1 Mahogany Log + 5 Nails + Coins)
- **Redwood Plank** (1 Redwood Log + 10 Nails + Coins)
- **Aether-treated Plank** (1 Aether Log + Void-Iron Rivets + Lumina)

---

## 5. Fletching & Engineering Consumables (The Log Sinks)
*What players actually do with tens of thousands of logs.*

- **Arrow Shafts** (1 Log = 15 Shafts. Scales up with better logs).
- **Shortbow / Longbow (Unstrung)** (1 Log = 1 Bow. Better wood = higher tier bow).
- **Crossbow Stock** (1 Log = 1 Stock).
- **Shield Base / Wooden Shield** (2 Logs).
- **Ship Hull Plating** (Requires massive amounts of Teak/Mahogany planks).
- **Magic Staff Base** (Requires Magic/Aether logs).

---

## 6. Skilling Outfits & Buffs
*Equippable items that increase the efficiency of the skill.*

- **Lumberjack Hat / Top / Legs / Boots** (Grants bonus Logging XP).
- **Nature's Sentinel Outfit** (Grants chance to chop 1 tick faster).
- **Woodcutting Cape** (Increases chance of finding Bird's Nests by 10%).
- **Rabbit Foot** (Amulet: Increases the chance of finding Bird's Nests).
