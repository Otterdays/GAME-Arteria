# Arteria: The AFK RPG

> [!WARNING]
> **ATTENTION:** Do NOT remove or delete existing texts, updates, docs, or anything else in this document. Only append, compact, or update.

Welcome to **Arteria**, a math-heavy, incremental idle RPG modeled after games like Melvor. 
Built on a modern React Native (Expo) architecture using Tamagui for beautiful, performant UI and Redux Toolkit for deep state management.

Check out the `DOCS/` folder for all architectural decisions, scratchpad, and tracking info.

- `DOCS/SUMMARY.md`: High-level overview.
- `DOCS/ARCHITECTURE.md`: Technical stack details.
- `DOCS/SCRATCHPAD.md`: Dev task queue.
